# Javier Llorente Portfolio

Portfolio personal de Javier Llorente, enfocado en experiencia, proyectos y recorrido profesional en producto digital.

## Stack

- Next.js
- TypeScript
- Tailwind CSS

## Funcionalidades

- Sitio one-page
- Idioma español / inglés
- Navegación lateral por secciones
- Modo oscuro / claro
- Diseño responsive

## Secciones

- Intro
- Enfoque
- Proyectos
- Momentos
- Recomendaciones
- Experiencia
- Contacto

## Instalación

```bash
npm install
npm run dev
